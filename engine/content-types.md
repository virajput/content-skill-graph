# Content Types

## Thread
A multi-post sequence for X that breaks one core idea into short, punchy connected posts.
- Best for: playbooks, step-by-step systems, breakdowns
- Structure: hook → steps → recap → CTA

## Carousel
A multi-slide visual post for Instagram or LinkedIn documents.
- Best for: frameworks, lists, transformation stories
- Structure: bold slide 1 hook → 5-8 supporting slides → CTA slide

## Short Take
A 2-4 line post with a sharp hook and one fast insight.
- Best for: X, Threads
- Structure: bold claim → context → punchline

## Long-Form Post
A detailed written post for LinkedIn, substack, or articles.
- Best for: narrative, case studies, behind-the-scenes insights
- Structure: hook → story → lesson → framework → CTA

## Video Script
A spoken-first format for TikTok, Reels, Shorts, or YouTube.
- Best for: demos, reactions, tutorials
- Structure: hook → setup → walkthrough → result → CTA

## Substack
A deep email format built around one strong idea.
- Best for: authority building and owned audience growth
- Structure: subject line → opening story → main lesson → framework → CTA
